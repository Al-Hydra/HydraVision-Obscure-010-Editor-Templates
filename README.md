# ObsCure 010 Editor Template
ObsCure 010 Editor Template is a template for the hex editor software 010 Editor.

With this template, you can open all ObsCure 1 and 2 formats (.zwo, .dic, .dip, etc.), analyze them, edit them, and better understand their bit format.

# Usage
Download the 010 Editor hex editor.

Go to Templates -> View Installed Templates -> Add, and select all .bt files extracted from the ObsCure 010 Editor Template.
